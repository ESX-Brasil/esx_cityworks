# esx_cityworks

Obras da cidade trabalho para ESX

# Primeiros passos

1. Add esx_cityworks para a pasta em recursos [esx]
2. Add em server.cfg
3. Add esx_cityworks.sql ao seu banco de dados

# Configuração

1. Configuração somente em config.lua
